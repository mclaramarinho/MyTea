Ao dar pull da main com novas alterações no banco de dados, siga o passo a passo abaixo para que não crashe!

1. Vá em View > SQL Server Object Explorer.
2. Procure o banco de dados com o nome MyTea
3. Exclua todo o banco de dados.
4. Apague todas as migrations e ApplicationDbSnapshot
4. add-migration
5. update-database
